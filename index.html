HoloLabel: AR user-in-the-loop online annotation tool for as-is building information

An AR scene segmentation tool that allows a user to annotate and segment building information while scanning it. We focus on classification using a predefined set of classes. The user can request for a segmentation proposal of the scene, and then manually adapt the segments using a virtual paint brush.

Contributions: Implementation of segmentation pipeline and algorithm, communication between HoloLens/Unity and python server

AVX2 Sphere Tracer

C implementation of a sphere tracer through implicit shapes, applying various optimizations using AVX2 vector instructions. The sphere tracer supports shadows, mirror and specular reflections, anti-aliasing, multiple light sources and 5 different types of object shapes.

Contributions: Implementation of reflections (mirror and specular) and anti-aliasing through supersampling. Implement vectorized geometry functions. Implement object-vectorization of distance function. Implement rendering and spheretracing for ray-vectorization.
